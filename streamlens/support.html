StreamLens Support

Need help setting up StreamLens? This page covers the basic setup steps, common fixes, and how to contact support.

Getting Started

StreamLens lets you use your iPhone as a wireless camera for OBS Studio. To use StreamLens, install the StreamLens desktop plugin, add StreamLens Device as a source in OBS, then open the StreamLens app on your iPhone.

Your iPhone and computer must be connected to the same Wi-Fi network.

Basic Setup

  1. Install the StreamLens desktop plugin on your Mac or Windows computer.
  2. Open or restart OBS Studio.
  3. Add StreamLens Device as a source in OBS.
  4. Open StreamLens on your iPhone.
  5. The app should find OBS automatically when both devices are on the same Wi-Fi network.
  6. Press LIVE in the app to start streaming.

If StreamLens Cannot Find OBS

  • Make sure OBS Studio is open.
  • Make sure StreamLens Device has been added as a source in OBS.
  • Make sure your iPhone and computer are on the same Wi-Fi network.
  • Allow OBS through your computer firewall if asked.
  • Avoid guest Wi-Fi networks, VPNs, or networks that block local device discovery.
  • Restart OBS and reopen the StreamLens app.

Windows Installation Help

On Windows, close OBS before installing the plugin. Copy the included data and obs-plugins folders into your OBS Studio folder.

For most users, the OBS Studio folder is:
C:\Program Files\obs-studio\

macOS Installation Help

On macOS, download the .pkg installer, open it, follow the installer steps, then restart OBS Studio.

The current macOS plugin build supports Apple Silicon Macs.

Subscriptions and Restore Purchases

StreamLens subscriptions are handled through Apple’s in-app purchase system. If you already subscribed, open Settings in the app and tap Restore Purchases.
