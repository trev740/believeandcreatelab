Installation

After installing, restart OBS and add StreamLens Device as a source.

macOS

  1. Download the macOS .pkg file.
  2. Open the package and follow the installer steps.
  3. Restart OBS Studio.
  4. Add StreamLens Device as a source.

Windows

  1. Download and unzip the Windows ZIP.
  2. Close OBS Studio completely.
  3. Copy data and obs-plugins into your OBS Studio install folder.
  4. Restart OBS Studio and add StreamLens Device as a source.
