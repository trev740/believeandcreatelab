When Reporting a Problem

Please include the following details so we can help faster:

  • Your device model, for example iPhone 14
  • Your iOS version
  • The AskOracle app version, if available
  • A short description of what happened
  • Steps to reproduce the issue, if possible

Voice Input Help

AskOracle uses microphone input and speech recognition so you can ask yes-or-no questions by voice.

If voice input is not working, please check that microphone and speech recognition permissions are enabled for AskOracle in your device settings.

Purchases & Restore

If you purchased Upgrade The Core and premium features are not showing, open the upgrade screen and use Restore Purchases.

After purchase is completed, the bolt icon changes to a crown icon, indicating that premium features are unlocked.
